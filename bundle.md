### Bundle模型对象
| 参数字段<img width=20/>  | 参数类型<img width=80/> | 是否必须   | 备注 |
|  ----   | ----  |----  |
| id | string    | 是  | 道具Id|
| name  | string | 否  | 道具名 |

| ctime  | string | 是  | 道具创建时间 |
| size  | long | 是  | 道具大小 |
